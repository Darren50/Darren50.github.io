SaveUrFood

This is a project that I have done for a school module.



Project Summary: Food wastage is a big issue among many countries. In 2011, around 1/3 of the world’s food was lost or wasted every year. When food is wasted, it puts pressure on resources as more food has to be sourced to meet the food demand for the country. Furthermore, more waste disposal facilities are required to handle wasted food. All of these can cause an increase in carbon footprint which can cause global warming. The root of the problem is people not finishing their food whether it is due to excess food or other reasons. The best way to prevent food wastage is to stop it from starting in the first place. This means that we should encourage people to buy what they need at all times and also give whatever they do not need to people who need it.

Generally, I feel that this app would be very compelling as the concept is simple to understand but also have plenty of cool features for users to use. Plenty of the time, food tracking apps don’t really become popular due to the nature of how tiring it can be to input data inside. However, due to the features that I have added such as AI Image to Text for names and the barcode add and remove rows that haven’t been tried in other food tracking apps, I believe that my app will stand out as it reduces the effort required to track food and also provides a more cool and convenient experience for my target segment, which are 25-54 years old who are decent at using technology.

Unique Features:

  • Community to allow others to find out info on other foodbanks
  • AI Image to text recognition for easier data input
  • Barcode scanner to input data and remove data easier when shopping
  • Expiry date calculator

Since this was my first time using Android Studio, I had to spend some time learning using Youtube videos. I first implemented the stuff like a splash screen and a login page using Firebase.

Afterwards, I decided to use the SQLite database to store what the food that the user inputted. Along the way, I implemented an expiry date calculator and AI recognition using google lens that can recognise text from image and input it automatically.

I also added a barcode sensor to scan barcodes with your phone. The barcode acts as an 'ID' which could be used later on for the user to remove all foods with the same barcode.

I added a 'Community section' for the user to visit several various foodbanks' websites as well.

Overall, this project was really fun as I get to dip my hands into app development which I have never done before. I do hope to get better at it someday!
